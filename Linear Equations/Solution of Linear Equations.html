Solution of Linear Equations - Various Connections covered

 

This section will contain examples of solving Linear equations. The methods demonstrated in these examples will cover 

  1. Using Equivalent fractions : Linear equations that are formed as ratios will be converted to equivalent fractions and solved using “ Visual Fractions” ( Like the whole-part diagrams that are introduced in Class 4/5.  The same problem is solved without the need for visuals.)
  2. A visual approach to using Similar triangles will be demonstrated. This example is to emphasize that ratio-proportion problems can be represented as similar triangles Class X (for two-variables).
  3. A method to convert Linear equations to be expressed as ratios is described. Post conversion of Linear equations to Ratios, the first method is used to solve them. Connect Class X to Class 7 /li>
  4. Using the Geometric defintion of a point -a point in two dimensions can be described as one that has fixed ratio of its distance from three lines, a method is for solving pair of linear equations is evolved. This method will find applications in Computer Vision, though not restricted to it.Connecting Geometry
  5. A brief understanding of the properties of linear functions, that is “Principle of Superposition” is introduced and the same will be applied to solving a pair of linear equations. This method will give a Segway to Vectors. Connecting Linear Equations (Class X) to Vectors (Class XII) 
  6. An ensemble of the above methods is used to extend the concept to Solution of equations in three dimensions. Connecting Ratio and Proportion (Class 7) to Linear Equations in 3 Variables (Class 11)
  7. Some of the above methods will be used to demonstrate the representation of Linear equations as a matrix, and the solution based on elimination method ( Reduced Row Echelon) is explored. Connections to Matrix (Class XI)
