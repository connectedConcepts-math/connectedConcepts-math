Using Superposition Principle

Introduction

The word “Superposition” principle is not explicitly stated in the text book. However the properties of this principle are used at various places. In view of avoiding using any terminology outside of the the text book,  the properties will be used.

The fundamental method used in this method is “Unitary Method”, as described in the Chapter on Ratio and Proportion in Class 7 of NCERT.

Unitary Method

The method in which we first find the value of one unit and then the value of the
required number of units is known as the unitary method. Suppose the cost of 6
cans is  ₹ 210. To find the cost of 4 cans, using the unitary method, we first find the
cost of 1 can. It is  ₹ \(\frac{210}{6}\). or ₹ 35. From this, we find the price of 4 cans as ₹ \(35*4 =140\)

Unitary Method Extended

Suppose a box of type A contains a bag of 5 Chocolates and another bag of 2 Biscuits.. Then if I have 5 boxes, then I have 5 bags each containing 5 chocolates, and 5 bags each containing 2 Biscuits.

Suppose some one tells me, I have 60 Chocolates and 24 Biscuits, and asks how many boxes of type A, do I have ?

Unitary method tells us that  there 12 boxes of type A.

Let us take a different problem:

 Each box of type B has two bags, first bag containing some number of chocolates and second bag containing some number of Biscuits. I have 7 boxes of type B., making me own  98 Chocolates and 84 Biscuits. My Friend has 56 Chocolates and 48 Biscuits. How many boxes of type B does he have ?

By Unitary method, one box of type B will have a bag of  \(98+7 = 14\) Chocolates in a bag, and \(84+7 = 12\) Biscuits in a bag. So, for 56 Chocolates and 48 Biscuits, the number of boxes should be \(56+14=4\) or \(48+12=4\) boxes.
